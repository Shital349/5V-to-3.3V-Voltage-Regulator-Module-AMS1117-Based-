# 5V-to-3.3V-Voltage-Regulator-Module-AMS1117-Based-
This project is a simple and reliable 5V to 3.3V power supply circuit designed using the AMS1117-3.3 linear voltage regulator. The circuit converts USB 5V input into a stable 3.3V output for powering microcontrollers, sensors, and embedded systems.  The design is suitable for robotics, IoT, drone electronics, and embedded hardware projects
